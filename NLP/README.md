# Проекты по обработке естественного языка (NLP)

Эта папка содержит учебные и исследовательские проекты в области NLP.

## Содержание

- **Распознавание речи (ASR)**:  
  - Применение модели `whisper-small` для транскрибации русскоязычных аудиозаписей  
  - Оценка качества через **WER** (Word Error Rate)

- **Исправление ошибок в транскрипциях**:  
  - Генерация датасета с помощью Groq API (или локальных LLM, например Ollama)  
  - Дообучение языковых моделей для постобработки выхода ASR  
  - Сравнение предобученных и дообученных моделей по WER

- **Оценка качества генерации текста**:  
  - **Perplexity** с использованием `ruGPT3-small`  
  - Семантическая метрика на основе **LSA + TF-IDF + SVD**

- **Эксперименты с моделями и токенизаторами**:  
  - Подбор наилучших ASR- и spell-correction моделей  
  - Тюнинг дообучения для максимизации качества

## Используемые инструменты

- Python, PyTorch, Transformers  
- Hugging Face (`whisper`, `ruGPT3-small`)  
- Groq API / Ollama (для генерации данных)  
- `torcheval` (WER), `scikit-learn` (TF-IDF, SVD)
