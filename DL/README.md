# Deep Learning Projects

Эта папка содержит проекты по глубокому обучению, выполненные в рамках курсов и самостоятельной практики.

## Содержание

- **YOLOv1 — реализация с нуля**:  
  - Кастомный датасет и формат разметки  
  - Архитектура на основе ResNet50 + голова детекции  
  - Реализация функции потерь в стиле YOLOv1  
  - Визуализация предсказаний

- **YOLOv11 (ультралёгкая версия)**:  
  - Быстрый старт с предобученной моделью через Ultralytics  
  - Инференс и оценка качества на валидационных данных

- **Языковые модели на RNN/LSTM**:  
  - Реализация авторегрессивной модели для генерации текста  
  - Обучение, инференс и управление через температуру  
  - Сравнение токенизаторов (BPE, unigram) по генерации и perplexity

- **Оценка качества генерации**:  
  - Perplexity через ruGPT3-small (Hugging Face)  
  - Семантическая оценка на основе LSA + TF-IDF + SVD

## Используемые инструменты

- Python, PyTorch, TorchVision  
- Hugging Face Transformers, Datasets  
- scikit-learn (SVD, TF-IDF)  
- Ultralytics (YOLOv11)  
- Matplotlib, OpenCV
