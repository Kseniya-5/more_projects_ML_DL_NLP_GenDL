# ML / DL / NLP / GenAI Projects

Этот репозиторий содержит подборку учебных и исследовательских проектов по машинному обучению, глубокому обучению, обработке естественного языка и генеративному ИИ.

## Структура
- **[ML](./ML)** — классические модели машинного обучения, EDA, регрессия, регуляризация, feature engineering  
- **[DL](./DL)** — глубокое обучение: YOLO, RNN/LSTM, генеративные модели  
- **[NLP](./NLP)** — распознавание речи, исправление ошибок, оценка качества через WER и perplexity  
- **[LLM](./LLM)** — языковые модели, токенизация, дообучение, оценка генерации

Все проекты выполнены в образовательных целях и сопровождаются ноутбуками, скриптами и описаниями.
