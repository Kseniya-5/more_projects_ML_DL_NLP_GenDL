# Machine Learning Projects

Эта папка содержит коллекцию учебных и исследовательских проектов по машинному обучению, выполненных в рамках курсов и самостоятельной практики.

## Содержание

- **Базовая обработка данных**: работа с пропусками, дубликатами, кодирование признаков, визуализация.
- **Разведочный анализ данных (EDA)**: анализ распределений, корреляций, связи признаков с целевой переменной.
- **Линейные модели**:
  - Линейная регрессия
  - Ridge, Lasso, ElasticNet
  - Подбор гиперпараметров с помощью `GridSearchCV`
- **Оценка качества**:
  - Метрики: $R^2$, MSE
  - Кастомные бизнес-метрики
  - Кросс-валидация и сравнение с лидербордом
- **Feature engineering**:
  - Кластеризация как источник признаков
  - Поиск аномалий (Isolation Forest, LOF, DBSCAN)
  - Использование кластеров и шума в качестве дополнительных признаков
- **Сервисы**:
  - Пример развертывания модели через FastAPI (prediction API)

## Используемые инструменты

- Python
- pandas, numpy, scikit-learn
- seaborn, matplotlib, missingno
- FastAPI, Pydantic
