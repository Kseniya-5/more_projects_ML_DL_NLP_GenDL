# Large Language Model (LLM) Projects

Эта папка содержит учебные и исследовательские проекты, посвящённые работе с большими языковыми моделями (LLM).

## Содержание

- **Обучение языковых моделей с нуля**:  
  - Реализация RNN/LSTM на PyTorch для генерации текста (авторегрессивная модель)  
  - Поддержка режимов обучения и инференса  
  - Управление генерацией через **температуру**

- **Сравнение токенизаторов**:  
  - BPE (с разным размером словаря)  
  - Unigram (пословный)  
  - Анализ влияния на качество генерации

- **Оценка качества генерации**:  
  - **Perplexity** через ruGPT3-small (Hugging Face)  
  - **Семантическая метрика** на основе LSA + TF-IDF + SVD  
  - Сравнение моделей по этим метрикам

- **Дообучение и исправление ошибок**:  
  - Генерация датасета для дообучения через Groq API (или локальные LLM, например Ollama)  
  - Дообучение модели для исправления опечаток в ASR-выходах  
  - Оценка качества через **WER** (Word Error Rate)

## Используемые инструменты

- Python, PyTorch  
- Hugging Face Transformers, Datasets  
- Groq API / Ollama (для генерации данных)  
- torcheval (WER), scikit-learn (TF-IDF, SVD)
